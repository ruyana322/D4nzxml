## Deploy ke Vercel (5 menit)

### 1. Upload ke GitHub
- Buat repo baru di github.com (Public)
- Upload semua file ini ke repo

### 2. Deploy ke Vercel
- Buka vercel.com → Sign in with GitHub
- Klik "New Project" → Import repo ini
- Klik "Deploy" — selesai!

### 3. Pakai Appnya
- Buka URL Vercel kamu
- Masukkan API Key dari replicate.com (gratis)
- Upload video → Enhance → Download!
